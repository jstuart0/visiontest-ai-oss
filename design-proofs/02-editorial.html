The slow art of watching your interface.

bythe VisionTest editors·8 min read

Most testing tools ask you to write code. This one asks you to write sentences. You describe a journey in plain English — log in, open the settings page, verify the banner is gone — and the machine takes it from there, frame by frame, step by step. It notices what you would have noticed, if only you had the time.

The premise is simple. The execution is not. Every step carries a confidence badge: exact, heuristic, unknown. Every run carries a goal. The goal is a contract, not a wish; unverifiable goals cannot be saved. The system refuses to lie to you.

What follows is a single run of a login scenario, annotated. The test passed. The baseline was updated. A human will read this before morning.

A testing tool that reads like prose, that runs like a darkroom, and that tells the truth about what changed. — Readme of record
