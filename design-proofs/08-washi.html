MATTERS · 能 目 · WHAT IT DOES
壱 · 01
Captures every screen さつえい

Chromium, Firefox, WebKit. Three viewports per device. iOS, Android, and the open web. A single image is taken carefully; thousands are taken without fuss.

弐 · 02
Compares with judgement ひかく

Pixel diff is the coarsest reading. SSIM for structure, LPIPS for perception, DINOv2 for semantics. The verdict comes last, and it is a single seal.

参 · 03
Quarantines the flaky 隔 離

A test that hesitates is set aside, not trusted, not thrown out. It is returned to the main register only when it has been steady for a while.

四 · 04
Proposes a quiet fix 修 繕

When the diff is plainly a bug, the model drafts a patch as a suggestion — never a commit — and leaves the decision, as always, with the human.
